## Getting Started

### Prerequisites

- [Claude Code](https://claude.ai/code) installed
- Access to Claude API (Sonnet 4)

### Usage

1. Open this repository in Claude Code
2. Provide a project description or URL to analyze
3. Run the analysis:

```bash
# The agents will automatically run in parallel
# Example command (from Claude Code):
"Analyze this SaaS project: [description or URL]"
```

4. Review the comprehensive reports generated by each agent

## Agent Capabilities

### SaaS Architect
- Technology stack evaluation and recommendations
- Cloud infrastructure strategy (AWS, Azure, GCP)
- Security and compliance architecture (GDPR, data residency)
- Scalability planning and cost analysis
- Integration architecture

### UI/UX Designer
- Target user persona research
- Competitive UX analysis
- User flow optimization
- Mobile responsiveness strategy
- Conversion rate optimization
- Accessibility compliance (WCAG)

### Backend Lead Engineer
- Database schema design (PostgreSQL, MongoDB, etc.)
- API architecture (REST, GraphQL)
- Authentication and authorization
- Third-party integrations
- Performance optimization
- Security best practices

### Frontend Lead Engineer
- Component architecture
- State management strategy
- Performance optimization

- Progressive Web App (PWA) recommendations

### Development Team Lead

- Competitive landscape assessment
